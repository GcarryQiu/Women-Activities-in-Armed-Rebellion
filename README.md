# Women Activities in Armed Rebellion

This project uses the Women’s Activities in Armed Rebellion (WAAR) dataset, which includes women’s participation in over 370 rebel organizations (1946–2015). The data include 22 measures covering women’s roles in non combat, combat, and leadership activities. The purpose of this project is to analyze the determinants of Female Leadership in Armed Groups? All contextual information are provided in the WAAR Handbook and the associated open access article in Research.
